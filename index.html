Unique HTML Tags

<map> — contextual image hotspots

map creates clickable areas on an image. In this demo, clicking the coffee cup redirects you to a related high-res close-up of coffee, while clicking the croissant takes you to a pastry close-up.

<canvas> — dynamic scriptable graphics

canvas acts as a resolution-dependent bitmap canvas, which can be used for rendering graphs, game visuals, or other visual images on the fly via JavaScript.

<s> — indicating irrelevant content

s is used for text that is no longer correct, accurate, or relevant. It visually strikes through the text but retains the semantic meaning of "no longer applicable."

<noscript> — fallback for disabled scripts

noscript provides an alternate way to display information for browsers that have JavaScript disabled or do not support client-side scripting.

Normally, you won't see anything here unless you disable JS in settings.

<svg> / <ellipse> — scalable vector ovals

ellipse is used to create an oval shape. Unlike a circle, you define two radii (rx and ry), allowing for flexible, perfectly sharp vector graphics at any zoom level.
